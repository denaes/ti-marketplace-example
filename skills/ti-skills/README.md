# Thought Industries (ti-skills)

Skills grounded in [Thought Industries](https://www.thoughtindustries.com/) public positioning: the **Customer Learning & Intelligence Platform** for creating, delivering, monetizing, and measuring training for **external audiences** (customers, partners, members, prospects) as well as employee L&D. Use them for positioning, scoping, and narrative alignment—not as a substitute for internal product docs or contracts.
